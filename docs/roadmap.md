# Roadmap

This page lists planned milestones only. It describes direction, not promises. For what ships today, read [getting started](getting-started/installation.md) and the [language reference](reference/syntax.md).

## v0.1 (current)

- `.nav` language: twelve directives, metadata, postings, Markdown bodies
- `nav serve`, `nav build`, `nav format`
- Local Reader and static-site export
- VS Code / Cursor extension: syntax highlighting and format-on-save
- Semantic diagnostics for common booking mistakes

## v0.2

- Language improvements from real-world workspaces
- Stronger validation and clearer diagnostics
- Documentation and examples aligned with stable booking patterns

## v1.0

- Stable specification for the documented language and CLI
- SemVer guarantees for breaking changes
- Migration guidance for any pre-1.0 workspaces

Feature ideas that are not listed here belong in discussions or RFCs, not in this roadmap.
