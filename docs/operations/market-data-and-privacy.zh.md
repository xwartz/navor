# 行情数据与隐私

Navor 不上传你的投资仓库，也不收集遥测、不要求注册账户，`nav serve` 只读本机你选定的路径，上下文怎么留在你手里，见 [AI 与 Navor](../concepts/ai.zh.md)。

价格和汇率是可选增强，不是源事实，Reader 会把缺失、过期或拉取失败标出来，不会把它们当成权威数据。

托管站点上，浏览器会向同源的 `POST /api/prices` 要行情，代理再把代码转发给配置的数据源。代码可能暴露你的持仓或关注标的，只有在条款、缓存策略和隐私说明都合适时，才值得部署代理。

Navor 帮你整理投资信息，不提供金融、法律、税务或投资建议。
