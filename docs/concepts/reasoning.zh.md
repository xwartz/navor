# 推理链

持仓回答**持有什么**，推理回答**为什么**。

Navor 按这条链来建模组合是怎么长出来的：

```text
Research（研究）
    ↓
Thesis（观点）
    ↓
Decision（决策）
    ↓
Transaction（交易）
    ↓
Portfolio（派生）
```

## 研究是证据

`research` 记的是你形成观点之前看到的外部事实，要能追溯来源（`source`），也要能回头翻。

## 观点可证伪

`thesis` 写清楚你相信什么，以及什么会推翻它（`invalid_if`），缺了这层，仓位就只剩数字。

## 决策是意图

`decision` 记你打算做什么、什么时候做、依据什么，真正成交另记在 `txn` 里。

## 交易是执行

`txn` 是已经发生的经济事实：买了多少、现金怎么动、手续费多少。

## 组合是派生

持仓和权重由交易和目标算出来，不是思考过程的权威记录。

## 只追加的演变

观点变了，就追加新的 `thesis`、`decision` 或 `review`，旧记录留着，Git 历史才能看出思路怎么变。
