# AI 与 Navor

大模型不知道你的持仓、观点和规则，投资问题就很难答到点子上。Navor 把这些事实写成结构化、人也能读的文档，正好补上这块上下文。

## 上下文问题

你问「今天要不要买 X？」，模型不知道你的仓位、目标、观点和过往失误，只能泛泛而谈。把 Navor 投资仓库交给它，就不用让它猜你的投资哲学。

## 模型能做什么

- 按标的归纳研究与观点
- 对照决策和声明的目标权重
- 找出有 `decision` 却没有对应 `txn` 的地方
- 用新的 `research` 去质疑既有 `thesis`
- 根据 `txn` 和正文解释历史记账

不用导出专有格式，也不用爬数据库。

## 人先读

你读不懂的文件，给 AI 也不是好上下文，Navor 先保证人读得懂，再谈机器怎么用。

## 可复现

纯文本放在 Git 里，你和模型看的是同一份快照，事实不会各说各话。

## 隐私

Navor 不上传你的投资仓库，粘贴、提交、部署都由你决定。若要托管站点或代理，请先读[行情数据与隐私](../operations/market-data-and-privacy.zh.md)。
